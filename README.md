# 📚 Curriculum Mesh 

An interactive curriculum based on the semesters of my major. This is a project that became a trend in Chile (at least) in 2025.

## 💻 Technologies

- `Vite`
- `React`
- `TypeScript`
- `Tailwind CSS`

## 💡 Features

- Use of localStorage to save user progress
- Definition of types for courses and semesters
- Simple yet aesthetic use of Tailwind for UI/UX and responsive design
  
## 🔻 Running the Project

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Open `http://localhost:5173` in your browser
